# 2YOU Agent Routing Matrix

## Purpose

This matrix preserves the full existing agent roster and assigns each agent a primary 2YOU role.

No agent is removed.
No expertise is discarded.
What changes is the routing discipline.

---

## Routing Logic by Layer

### Layer 1: Strategy
Agents used to define where the business should go.

### Layer 2: Tactic
Agents used to convert the strategy into executable plans, structures, assets, and sprints.

### Layer 3: Operation
Agents used to execute work.

### Layer 4: Orchestration
Agents used to coordinate, govern, validate, and re-route work.

## Activation Rules by Type of Request

### Offer / Positioning / Proposal Request
1. 2YOU Context Keeper
2. 2YOU Chief Orchestrator
3. Studio Producer or Proposal Strategist or Brand Guardian
4. Senior Project Manager
5. downstream execution only if approved

### Funnel / Campaign / Launch Request
1. 2YOU Context Keeper
2. 2YOU Chief Orchestrator
3. Growth Hacker / Social Media Strategist / Sprint Prioritizer
4. Project Shepherd
5. execution specialists
6. Analytics + QA loop

### Build / Product / Platform Request
1. 2YOU Context Keeper
2. 2YOU Chief Orchestrator
3. Studio Producer / Senior Project Manager / UX Architect
4. Agents Orchestrator
5. engineering specialists
6. QA and reality layers

### Recovery / Chaos / Misalignment Request
1. 2YOU Context Keeper
2. 2YOU Chief Orchestrator
3. Project Shepherd
4. Workflow Optimizer
5. Agents Orchestrator
6. corrected functional owners

---

## Special Rule for Autonomous Content Engines

Agents such as Carousel Growth Engine, Content Creator, TikTok Strategist, Instagram Curator, and other high-output specialists must not self-activate from raw enthusiasm.

They require:
- objective
- offer
- ICP
- CTA
- KPI
- mode
- channel role in the funnel

Without those, they should be blocked and redirected to the tactical layer.
