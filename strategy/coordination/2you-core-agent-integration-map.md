# 2YOU Core Agent Integration Map

## Purpose

This file explains how the new 2YOU layer connects to the four core legacy agents that drive most system behavior:
- Agents Orchestrator
- Studio Producer
- Senior Project Manager
- Project Shepherd

These agents are not removed.
They are re-ordered and re-scoped.

---

## Integration Sequence

### Step 1: Governance
- 2YOU Chief Orchestrator classifies the request
- 2YOU Context Keeper provides current business truth

### Step 2: Strategic Framing
- Studio Producer validates initiative role, priority, and business logic

### Step 3: Tactical Structuring
- Senior Project Manager converts approved direction into realistic scoped work
- Sprint Prioritizer sequences when needed

### Step 4: Execution Coordination
- Project Shepherd coordinates cross-functional streams
- Agents Orchestrator runs pipeline logic for multi-agent workflows

### Step 5: Validation and Learning
- QA / Reality / Analytics agents validate output
- 2YOU Context Keeper absorbs new truth

---

## Precedence Rules

### Rule A
If 2YOU Chief Orchestrator and a legacy orchestration agent conflict, 2YOU Chief Orchestrator wins.

### Rule B
If current initiative memory conflicts with a stale brief, 2YOU Context Keeper wins.

### Rule C
If strategic direction is missing, Senior Project Manager and Project Shepherd should escalate rather than improvise.

### Rule D
If a workflow is operationally complete but commercially misaligned, Reality and Analytics layers should block scale.

---

## Why this matters

Without this integration, the system risks becoming:
- strategically loose
- operationally noisy
- commercially inconsistent
- excellent at producing work nobody actually needed

That combination is more common than people admit, largely because it can be mistaken for productivity from a distance.
